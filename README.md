# UCLan App – Interactive UI/UX Prototype

An interactive UI/UX prototype for a university mobile app designed to improve the student experience at UCLan.

---

## 🎨 Figma Prototype

👉 Live prototype:  
[Open UCLan App Prototype](https://www.figma.com/proto/EzuV0HFlIhix4Lx5m9mHFI/Uclan-App.-Unity?node-id=1-14&starting-point-node-id=7%3A148&t=J3RR2n78wq0lRdil-1)

---

## 🧩 Key Features

- Home screen with quick access to key services
- Timetable and schedule overview
- Notifications and important updates
- Easy navigation to core student tools

---

## 🧠 Design Process

- Identified core student needs and pain points
- Created low-fidelity wireframes to explore layout and structure
- Designed high-fidelity screens in Figma
- Built interactive flows to simulate real app behaviour

---

## 🛠 Tools Used

- Figma
- UI/UX Design
- Interaction Design
- User-centred design principles
