# UCLan App Prototype 📱

Interactive UI/UX prototype for a comprehensive university mobile application designed in Figma, featuring intuitive navigation and modern design principles.

## 📋 Overview

UCLan App Prototype is a feature-rich mobile application prototype designed for:
- **Student Portal**: Access courses, grades, and schedules
- **Campus Navigation**: Interactive maps and directions
- **Event Management**: Discover university events and activities
- **Library Services**: Browse and book library resources
- **Campus Facilities**: Book rooms and facilities
- **Student Community**: Connect with peers and collaborate

## 🎯 Key Features

✅ Student authentication and profiles  
✅ Course management and schedules  
✅ Grade tracking and transcripts  
✅ Interactive campus map  
✅ Event listings and registration  
✅ Library resource management
